# 🚀 Dream Guiders

A smart and interactive **career guidance web application** that helps students explore the right career paths, plan their studies, and get instant guidance through an AI-powered chatbot.

---

## 🌐 Overview

Dream Guiders is designed to simplify career decisions for students after 10th and 12th. It provides structured guidance, useful tools, and an engaging UI to make the process easy and effective.

---

## ✨ Key Features

* 🔐 **User Authentication**

  * Secure Login & Signup system

* 🎨 **Dynamic Themes**

  * Multiple UI themes (Dark, Light, Ocean, Forest, Sunset, Rose)

* 🤖 **DreamBot (Chatbot)**

  * Instant career guidance and support

* 🎯 **Career Guidance System**

  * Stream selection after 10th
  * Career options after 12th

* 🏫 **College Finder**

  * Explore colleges based on interests

* 📝 **Resume Builder**

  * Create professional resumes easily

* 📅 **Study Planner**

  * Track tasks and maintain study consistency

* 🔖 **Bookmarks**

  * Save important content for later

* ❓ **FAQ Section**

  * Quick answers to common questions
